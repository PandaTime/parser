Получить экскель файл можно кликнув сюда
Чтобы добавить новые записи нужно:
  1. Установить Chrome Extension
  2. Открыть этот линк(на других работать не будет)
  3. Добавить во вкладку "Update IDS List" список айдишек*
  4. Пройти капчу
  5. Нажать "Parse Page"
  6. Нажать "Next Page"
Примечания:
italic = опциональный шаг *Прим.: каждый раз нажимая "Parse Page" с него будет убирать текущая айдишка.
