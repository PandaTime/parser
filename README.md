## How to:
1. В хроме перейти на : chrome://extensions
2. В выбрать "Developer Mode"
3. Добавить экстеншин(папку "chrome_extension")
4. Запустить сервер (python server.js || make start в папке "server")
5. Перейти на localhost:8080<br>
...<br>
6. Profit!<br>
